# Library Management System

## Overview

Full-stack library management system with search, lending, admin analytics, and AI discovery.

## Local Development

- Backend: `backend/`
- Frontend: `frontend/`

### Environment

Copy `.env.example` files in `backend/` and `frontend/` and fill values.

### Run (expected tooling)

- Backend: `npm install` then `npm run start:dev`
- Frontend: `npm install` then `npm run dev`
